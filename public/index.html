BeamNG.drive Multiplayer Prototype

This app provides a lightweight relay server and BeamNG Lua sample script so multiple players can publish and consume vehicle transforms.

Quick Start

  1. Run npm start.
  2. Create a host session with POST /api/sessions.
  3. Join with POST /api/sessions/:id/join.
  4. Each client sends its transform to POST /api/sessions/:id/state.
  5. Each client polls GET /api/sessions/:id/state?playerId=<id>.

Example Create Session

curl -X POST http://localhost:3010/api/sessions \
  -H 'Content-Type: application/json' \
  -d '{"hostName":"Host","mapName":"west_coast_usa"}'
